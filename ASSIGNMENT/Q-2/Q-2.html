2. WHAT IS THE USE OF isNaN FUNCTION?

INTRODUCTION:


    - NaN is stands for "Not-a-Number".

    - isNaN() Function is used to check whether a given value is illegal number or not.

    - The isNaN() method returns true if a value is NaN(Not-a-Number).

SYNTAX:
isNaN( value );

Here,
    value: Value passed in the isNaN() function.
    Return value: It returns a Boolean value.

EXAMPLE:
<script>
    console.log(isNaN(12));
    console.log(isNaN(0/0));
    console.log(isNaN(12.3));
    console.log(isNaN("Miral"));
    console.log(isNaN("01/07/2004"));
    console.log(isNaN(-46));
    console.log(isNaN(NaN));
</script>

OUTPUT:
false
true
false
true
true
false
true
