WHAT IS jQuery?

  ◊   jQuery is library of JavaScript.

  ◊   jQuery is Programming Language.

  ◊   jQuery is lightweight Scripting Language.

  ◊   Lightweight means, many lines of JavaScript code, wraps them into methods that can you call
       into a sigle line is called lightweight Scripting Language.

  ◊   jQuery is client side Language.

  ◊   The purpose of jQuery is "write less, do more."

  ◊   jQuery is called through the CDN (CDN-Content Delievery Network) or library file.

  ◊   jQuery is browser independent.

  ◊   jQuery extension is [.js]

  ◊   We can Apply jQuery:
        1. <head>
        2. <body>
        3. <script> (External file)

SYNTAX:

1.  $(document).ready(function(){
        //statements;
      });

        //statements;
