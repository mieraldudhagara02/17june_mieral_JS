5. WHAT DO YOU KNOW ABOUT THE BOOTSTRAP GRID SYATEM?

Bootstrap's Grid System is built with flexbox.
Grid System is fully responsive.
Bootstrap Grid System allows up to 12 columns across the page.
You can use all combinations of values summing up to 12.
