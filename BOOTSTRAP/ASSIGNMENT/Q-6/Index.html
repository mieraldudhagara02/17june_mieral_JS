6. WHAT IS THE DIFFERENCE BETWEEN BOOTSTRAP 4 AND BOOTSTRAP 5?

BOOTSTRAP 4 BOOTSTRAP 5
Bootstrap 4 is newer version of Bootstrap. Bootstrap 5 is newest version of Bootstrap.
It has jquery and all related plugins. Jquery is removed and switched to vanilla JS with some working plugins.
Bootstrap 4 supports both IE 10 and 11. Bootstrap 5 doesn't support IE 10 and 11.
Bootstrap 4 provides a few colors for a web page. Extra colors insert into the looks, and the card's color palette was upgraded.
There are a variety of colors to pick from the Bootstrap 5 framework.
Bootstrap 4 uses fewer CSS properties. Bootstrap 4 uses more CSS properties.
