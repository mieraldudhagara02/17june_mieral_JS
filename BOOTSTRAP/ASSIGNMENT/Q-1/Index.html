1. WHAT ARE THE ADVANTAGES OF BOOTSTRAP?
INTRODUCTION:
Bootstrap is managed and developed by "Twitter."
Bootstrap is the most popular HTML, CSS, and JavaScript framework.
Bootstrap is completely free to download and use!
ADVANTAGES:
1. Easy to use:
If you have a basic knowledge of HTML and CSS, you can use Bootstrap easily.
2. Responsive Feature:
People wants to browse responsive website. Through Bootstrap we can browse responsive website. A responsive design will automatically resize to suit the page.
3. Grid:
Grid is the major advantage of Bootstrap. It's fully flexible to maintain containers, raws and columns.
4. Browser Independent:
Bootstrap is compatible with all morden Browser.
