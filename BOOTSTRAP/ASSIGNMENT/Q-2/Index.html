2. WHAT IS BOOTSTRAP CONTAINER, AND HOW DOES IT WORKS?
Container is used to content raws and columns inside them.
There are two Container classess:
.container:

It's width (max-width) will change on different screen size.
The .container provides responsive fixed width container.

.container-fluid:

If you want to create full width (100%) container, then you have to use .container-fluid
The .conatiner-fluid provides full width container.
